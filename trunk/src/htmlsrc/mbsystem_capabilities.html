Short List of MB-System Capabilities:

  • General Documentation
    • MB-System manual page: mbsystem
    • MB-System I/O library manual page: mbio
  • Version 5 Programs and Macros:
    • These are the programs and macros that are used in the version 5 parallel processing scheme. Those programs that are also relevant to the older serial processing scheme are marked with an *.
    • General Functions
      • List swath data formats supported by MB-System: mbformat*
      • Get swath data file information and statistics: mbinfo*
      • Produce arbitrary ascii tables from swath data: mblist*
      • Copy swath data with optional time/space windowing and format change: mbcopy*
      • Translate groups of swath data files between formats: mbm_copy
      • Translate groups of swath data files between formats using parallel processes: mbm_multicopy
      • Set MB-System default parameters: mbdefaults*
      • List files referenced in a recursive datalist file tree: mbdatalist
      • Generate ancilliary files used in data managment: mbdatalist
      • Generate ancilliary files used in data managment using parallel processes: mbm_multidatalist
      • Generate an MB-System datalist file referencing all identifiable swath files in the specified target directory.: mbm_makedatalist
      • Create or modify parameter files used to control mbprocess: mbset
      • Apply specified processing tasks to a swath data file: mbprocess
      • Apply specified processing tasks to a swath data file using parallel processes: mbm_multiprocess
      • Convert between calendar time and unix time: mbtime
    • Water Properties Utilities
      • Obtain reference water sound velocity profile (SVP): mblevitus*
      • Calculate water sound absorption from water properties: mbabsorption
      • List CTD data in swath sonar data files: mbctdlist
      • Process data from a Seabird SBE53 pressure sensor into a tidal model for use by mbprocess: mbm_bpr
      • Generate a sound velocity profile model from sound speed and depth values in swath files: mbm_makesvp
    • Navigation Data Processing
      • Interactive navigation editor: mbnavedit
      • List navigation from ping records in a swath data file: mblist
      • List navigation from navigation records in a swath data file: mbnavlist
      • Create or modify parameter files used to control mbprocess: mbset
      • Merge navigation with swath data file (among other tasks): mbprocess
      • Adjust navigation to match bathymetric features in overlapping swathes: mbnavadjust
      • Merge inertial and surface tracking navigation for submerged sonar platforms (e.g. ROVs and AUVs): mbauvnavusbl
      • Output a list of the times when a survey passes the waypoints of a planned survey route: mbroutetime
    • Patch Test Analyis
      • Interactive 3D visualization data cleaning and attitude bias patch test: mbeditviz
    • Bathymetry Data Processing
      • Automatic bathymetry data cleaning: mbareaclean
      • Automatic bathymetry data cleaning: mbclean
      • Interactive bathymetry data cleaning: mbedit
      • Interactive 3D visualization data cleaning and attitude bias patch test: mbeditviz
      • Extract bathymetry edits in edit save file format: mbgetesf
      • Extract bathymetry edits from Simrad Neptune software into edit save file format: mbneptune2esf
      • Obtain reference water sound velocity profile (SVP): mblevitus*
      • Extract water sound velocity profiles (SVPs) from data: mbsvplist
      • Interactive modeling of SVP effects on bathymetry: mbvelocitytool*
      • Create or modify parameter files used to control mbprocess: mbset
      • Recalculate or correct swath bathymetry and apply bathymetry edits from mbedit and mbclean: mbprocess
      • Obtain histogram of bathymetry data: mbhistogram*
      • Automatically grid bathymetry data: mbm_grid*
      • Grid bathymetry data: mbgrid*
      • Extract subarea of GMT grid file: mbm_grdcut*
      • Perform forward and inverse UTM projections of ASCII data triples: mbm_utm*
      • Estimate attitude time lag by cross correlation of apparent bottom slope with the roll time series: mbrolltimelag
    • Sidescan and Amplitude Data Processing
      • Obtain grazing angle correction tables from sidescan/amplitude data for use by mbprocess: mbbackangle
      • Create or modify parameter files used to control mbprocess: mbset
      • Correct sidescan/amplitude data using tables generated by mbbackangle: mbprocess
      • Automatically mosaic sidescan/amplitude data: mbm_grid*
      • Mosaic sidescan/amplitude data: mbmosaic*
      • Grid sidescan/amplitude data: mbgrid*
      • Extract subarea of GMT grid file: mbm_grdcut*
    • Grid Vizualization
      • GMT grid vizualization and survey planning: mbgrdviz*
    • Survey planning
      • GMT grid vizualization and survey planning: mbgrdviz*
      • Translate an mbgrdviz survey route file into an MBARI AUV mission script: mbm_route2mission*
    • Postscript Graphics
      • GMT compatible swath contouring: mbcontour*
      • GMT compatible swath color fill plotting: mbswath*
      • Macro to generate GMT map of swath data: mbm_plot*
      • Macro to generate GMT map of gridded data: mbm_grdplot*
      • Macro to generate GMT perspective view of gridded data: mbm_grd3dplot*
      • Macro to generate GMT plot of xy data: mbm_xyplot*
      • Macro to generate GMT histogram plot of data: mbm_histplot*
    • Exporting Data Products to GIS and Visualization Packages
      • Generate GeoTIFF image from GMT grid files: mbgrdtiff*
      • Macro to generate GeoTIFF image of gridded data: mbm_grdtiff*
      • Convert GMT grid file to ArcView ASCII grid file: mbm_grd2arc*
      • Convert ArcView ASCII grid file to GMT grid file: mbm_arc2grd*
      • Convert GMT grid file to a set of tiled GeoVRML files: mbm_grd2geovrml*
    • Subbottom profiler (and center beam reflection) data processing
      • Extract subbottom profiler data to segy files: mbextractsegy
      • Get segy data file information and statistics: mbsegyinfo
      • Produce arbitrary ascii tables from segy data files: mbsegylist
      • Grid seismic data from segy data files: mbsegygrid
      • Generate Power Spectral Density (PSD) estimates from segy data files: mbsegypsd
    • Tide modeling and correction
      • Generate open ocean tidal model using the Oregon State Ocean Tide Prediction Software (OTPSnc) package: mbotps
    • Format-specific processing, extraction, or translation
      • Preprocess Reson 7k data (format 88), including applying time lag and biases to attitude and navigation data: mb7kpreprocess
      • Preprocess data from third generation Kongsberg multibeam sonars (e.g. EM122, EM302, EM710): mbkongsbergpreprocess
      • List raw information contained in Hydrosweep DS data files (formats 21 and 24): mbhsdump
      • Preprocess HYSWEEP HSX data (format 201), including applying time lag and biases to attitude and navigation data: mbhysweeppreprocess
      • Scan a Hydrosweep swath sonar data file using the program mbhsdump and generate a list in columnar format with time, date, latitude, longitude, C-mean, and C-keel entrys: mbm_fmtvel
      • Extract Jstar format (format 132) sidescan and subbottom data from Reson 7k (format 88) data files: mb7k2jstar
      • Extract Edgetech sidescan sonar data from Reson 7k format data, bin and lay the sidescan onto the seafloor, and output data in the MBF_MBLDEOIH formst (MBIO format id 71): mb7k2ss
      • mbauvloglist: Generate arbitrary text lists of values from MBARI AUV mission log files.
